Controls:
- W/S - move forwards or backwards
- A/D - rotate the ship
- C - stop moving
- Shift + A/D - move in side direction
- LMB - shoot in direction of cursor

Goal of th game:
- Recharge all beacons
- Kill enemies
- Collect gears
